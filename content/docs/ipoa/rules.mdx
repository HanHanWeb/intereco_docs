# 大赛总则



### 大赛目的

界面生态 PPT OS 创意设计大赛（Intereco PowerPoint OS Awards，简称 IPOA 赛事）由界面生态主办，系公益在线赛事，活动旨在为圈内创作者提供设计交流展示的优质平台，孵化 PPT OS 设计领域的创意人才与优质作品，提升界面生态知名度与圈子影响力，丰富社区生态内容，增强社区凝聚力，推动 PPT OS 圈子的创新与发展。



### 大赛原则

1. 公平公正公开原则：所有参赛作品评选标准统一、评选流程透明，评选结果及时公示，接受社区全体成员监督，杜绝任何暗箱操作、徇私舞弊等影响大赛公平性的行为。

2. 公益自愿原则：大赛全程不收取任何报名、评审、参赛费用，不涉及任何商业变现。参赛选手自愿报名、自愿提交作品。

3. 创意导向原则：重点关注作品的原创性、创意性，兼顾视觉表现与交互实用性，鼓励在遵循基本设计原则的基础上，大胆尝试，展现自身潜力。

4. 规范有序原则：严格遵循大赛规则开展各环节工作，明确时间节点、评审标准、奖项设置，确保大赛流程顺畅、正式规范。

5. 章程统领原则：章程是 IPOA 赛事一贯遵循的基本准则，赛事各项活动开展必须严格按照章程规定执行，鼓励在章程规定框架内对赛事细则进行创新、完善。

6. 文件规范原则：大赛各类配套文件的编写须遵循章程规范，不得违背章程内容与章程精神。

7. 民主修订原则：章程的修订须经界面生态运营团队与大赛组委会通过民主程序审议通过。

8. 口径统一原则：赛事对外沟通、宣传推广工作的正式口径须以章程及相关规定为依据，避免造成认知错乱与规则曲解。

9. 信息公开原则：赛事中涉及参赛者知情权范畴的相关文件与资料，依照对应公开办法予以公开。



### 组织单位

主办：界面生态

承办（执行）：IPOA 赛事组委会（以下简称大赛组委会）



### 参赛对象

面向中文互联网所有 PPT OS 设计的相关爱好者、团队与界面生态社区成员开放。



# 参赛要求



### 作品要求

1. 作品主题：参赛作品聚焦 PPT OS 创意设计，进行系统界面视觉设计、交互逻辑规划、功能概念创新等方向，要求内容积极健康、贴合实际使用场景，无低俗、违规、抄袭等不良内容。

2. 作品类别：可提交的作品类型包括：原创作品、改编作品。在提交作品时，参赛选手（或团队）应当选择适当的类型。其中，原创作品必须为参赛选手（或团队）原创，并在未侵犯第三方的知识产权、著作权、肖像权等合法权益前提下，自主设计、创作的作品。改编作品为参赛选手（或团队）在未侵犯第三方的知识产权、著作权、肖像权等合法权益前提下，基于现有设计进行自主改编所产生的衍生作品。

3. 公益约定：本次大赛为公益性质，参赛作品一经提交，即视为授权界面生态与大赛组委会在社区官网、社群、宣传物料等渠道进行非商业性展示、推广、交流使用，作者保留作品署名权，社区也不会将作品用于任何商业盈利活动。



### 报名及作品提交要求

1. 报名与提交方式：参赛选手（或团队）需在期限内通过大赛官网的报名入口，依照公告中的指定格式，如实提交 PPT OS 设计作品源文件，作品截图、作品名称、作品类型、作品的版本号、提交版本的完成日期、作品介绍、署名作者（个人或组织）、作者（团队负责人）联系方式、创作环境、鼓励搭配设计说明文档（可选提交），其中，改编作品需要按照指引，单独提交原型作品有关信息。作品提交后仅能修改部分内容，不可撤回；进入初审阶段后不可提交、删除、修改、替换，若有特殊情况需联系大赛组委会沟通。

2. 提交规范：作品必须按照活动公告的指定格式如实填写信息，未按格式提交、提交材料不完整、提交内容存在失实、违规等情形的，视为无效参赛。

3. 报名及提交须知：每人（或每队）仅可提交1件作品，不可重复报名、提交多件作品。



# 大赛流程



本次大赛全程在线开展，各环节时间节点将在界面生态社区官网（intereco\.org\.cn）、社群提前公示，逾期未完成对应操作的，视为自动放弃参赛资格。具体流程如下：



### 报名及作品提交阶段（10 至 20 天）

1. 大赛启动：在界面生态社区官网（intereco\.org\.cn）、社群、社交媒体平台发布大赛公告，明确规则、流程、奖项、报名信息填写格式，启动报名通道。

2. 报名通道开放：参赛选手（或团队）填写信息并提交作品，大赛组委会同步开展报名信息及作品初步核查（核查信息真实性、作品格式合规性）。

3. 提交截止：报名及作品提交通道关闭，不再接受任何报名及作品提交；大赛组委会整理参赛作品，核对参赛资格，剔除无效作品（格式不符、材料不全、信息虚假等）。



### 作品初审阶段（共 1 天）

1. 初审小组：由圈内知名创作者、界面生态核心创作者组成初审小组。

2. 初审流程：初审小组对所有有效参赛作品按照评分标准要求，完成通用基础评分，筛选出符合大赛要求、具备一定创意和质量的作品，进入复审及"人气之星"投票阶段。未通过初审的作品不进入复审及"人气之星"投票阶段，不参与最终评奖。



### 复审及"人气之星"投票阶段（共 3 天）

1. 复审小组：由圈内知名创作者、界面生态核心创作者组成复审小组，确保评审专业性。

2. 复审流程：复审小组对所有初审通过的作品按照评分标准要求，完成**分类特色评分**，重点考核作品创意性、视觉表现、交互概念、实用性等。

3. "人气之星"投票：进入复审阶段同时，大赛官网同步开放社区投票通道（仅界面生态社区注册用户可参与投票），将展示所有初审通过的作品，社区成员可为喜爱的作品投票，每人每日限投 1 票，共计最多可投 3 票，严禁刷票（经监测、认定存在刷票行为的，将取消有关投票资格，并视情形对作品参赛资格进行调整）。

4. 投票截止及统计：投票通道关闭，大赛组委会统计投票结果，得票最高的作品获得"人气之星"奖（若得票相同，结合最终评分确定）；初审阶段评分、复审阶段评分结果及投票结果整理归档，进入终审阶段。



### 终审阶段（共 1 天）

1. 终审小组：由圈内知名创作者、界面生态核心创作者、界面生态运营团队代表组成终审小组，保障终审权威性。

2. 终审流程：终审小组需结合作品实际表现，校验初审、复审的评分与评分意见，若确认评审存误的，在终审意见中更正错误并计算最终得分；若确认无误的，由前两次评分的简单相加计算出最终评分。终审最后，按照最终评分与投票结果（仅"人气之星"奖），确定所有奖项的获奖名单，确保奖项评选公平、公正、贴合大赛原则。

3. 终审核查：终审工作中，若发现作品存在违规行为，立即取消获奖资格。空缺奖项可由终审小组从候选作品中补选，或取消该奖项。



### 结果公示阶段（共 3 天）

1. 结果公示：在界面生态社区官网（intereco\.org\.cn）、社群公示所有获奖名单，标注各奖项获奖选手（或团队）、作品名称，公示期间接受全体社区成员及公众监督。

2. 异议处理：公示期间，若对获奖结果有异议，可向大赛组委会提交书面异议及相关证明材料，大赛组委会在当日内核实处理，反馈处理结果；公示无异议，获奖名单正式生效。



### 作品展示阶段

1. 奖项发放：获奖公示结束后，大赛组委会联系获奖选手（或团队），发放电子荣誉证书；若有实物奖励，同步安排发放。

2. 作品展示：在界面生态社区官网（intereco\.org\.cn）开设优秀作品展示专区，展示所有获奖作品及部分优秀入围作品，附上设计说明及评审点评，供社区成员交流学习。

本次大赛奖项设置兼顾个人与团队、专业与人气、新成员与资深成员，所有奖项均为公益荣誉性质，可结合社区资源设置对应权益奖励，具体如下：

|综合奖项|一等奖|电子荣誉证书、社区高级周边、创作激励奖金、优秀作品重点展示、社区认证|占通过初审作品总数 10%|

### 核心评选标准

1. 基础标准：作品符合大赛参赛要求，内容积极健康。

3. 合理性：界面布局合理、色彩搭配协调，视觉效果美观，符合大众审美及 PPT OS 使用场景。

4. 交互概念：交互逻辑清晰、合理，具备实用性和可操作性，交互概念有创新性，贴合用户使用习惯。

5. 创意性：设计理念新颖，创新技术与运用形式，具备独特的创意表达和创新思维。

5. 综合表现：设计说明完整、思路清晰，作品整体完整性高，具备一定的推广价值和学习意义。



备注："人气之星"奖仅以社区投票结果为评选依据；"黑马突围"奖结合参赛选手（或团队）历史参赛情况、作品质量提升幅度、最终评分综合确定。



# 评审委员会组成及职责



### 评审委员会组成

1. 初审小组：由自愿报名的圈内知名创作者、界面生态核心创作者组成初审小组。

2. 复审小组：由自愿报名的圈内知名创作者、界面生态核心创作者组成复审小组。

3. 终审小组：由自愿报名的圈内知名创作者、界面生态核心创作者与界面生态运营团队代表组成终审小组。



### 评审职责

1. 严格遵循大赛评审标准及原则，公平、公正、公开地开展评审工作，不偏袒、不徇私，不接受参赛选手的宴请、馈赠等，独立完成评分及评审意见。

2. 严格执行评委回避原则，不参与同自身相关联，影响公平、公正、公开开展评审工作的作品评审。

3. 认真审核每一件参赛作品，重点核查各项要求，及时发现违规作品并反馈大赛组委会，剔除无效作品。

4. 评审过程中做好记录，对获奖作品给出简要评审点评，为社区成员提供学习参考。

5. 严格遵守评审纪律，不泄露评审身份、评审过程、评审分数、候选名单等未公示信息，确保评审工作的严肃性。

6. 参与异议核实工作，结合专业判断，给出合理的处理意见，反馈给大赛组委会。



# 纪律要求



1. 参赛选手（或团队）需严格遵守大赛规则及各项要求，如实填写报名信息，提交原创作品，严禁虚假报名、抄袭、刷票、作弊等违规行为；违规者取消参赛资格、获奖资格，情节严重者将在社区公示，限制社区相关权益。



2. 评审委员会成员需严格遵守评审纪律，坚守公平公正原则，严禁泄露评审信息、徇私舞弊，若有违规行为，立即取消评审资格，追究相关责任。



3. 大赛工作人员需认真履行职责，做好报名接待、作品整理、评审组织、结果公示、异议处理等工作，热情服务参赛选手，严禁推诿、敷衍，确保大赛顺利开展；严禁利用工作之便谋取私利、泄露参赛选手个人信息。



4. 所有参与大赛的人员（参赛选手、评审、工作人员）需尊重他人劳动成果，遵守社区规则，文明交流，严禁出现辱骂、诋毁、攻击他人的行为，营造良好的大赛氛围。



# 异议处理机制



1. 异议提交：终审结果公示期间，任何单位或个人对结果有异议的，可向大赛组委会提交书面异议（需注明异议人姓名、联系方式、异议理由、相关证明材料），逾期未提交的，视为无异议，不再受理。



2. 异议核实：大赛组委会在收到异议后，1个工作日内组织相关人员（初审/终审小组）进行核实，结合作品实际情况、评审记录，给出客观公正的处理意见，确保异议处理公平、透明。



3. 结果反馈：大赛组委会在核实完毕后，1个工作日内将处理结果反馈给异议人，说明核实过程及处理依据；若异议成立，将调整相关结果（如取消违规作品资格、重新评选候选名单），并在社区官网补充公示；若异议不成立，将详细说明理由，异议人不得再次以同一理由提交异议。



4. 异议免责：异议人需对提交的异议及证明材料的真实性负责，严禁捏造、歪曲事实、诬告陷害他人，若发现虚假异议，将追究相关责任，情节严重者限制社区相关权益。



# 赛事文件管理

### 赛事文件层级与效力

IPOA 赛事文件体系以本章程为核心，由章程、配套文件两级构成，各级文件效力与定位如下：

1. 章程为最高准则：本章程是赛事一贯遵循的基本准则，赛事全流程活动开展、各类文件制定，均须严格以章程规定为依据，不得违背章程内容与核心精神；鼓励在章程框架内对赛事执行细则进行创新、优化与完善。

2. 配套文件为执行补充：配套文件是对章程条款的细化、补充与操作指引，效力低于章程，不得与章程规定相抵触；若配套文件与章程内容不一致，以本章程为准。



### 章程修订管理

#### 修订原则

1. 民主决策、公开透明：充分听取社区各方意见，保障修订程序公正合规。

2. 审慎稳定：仅在赛事发展确有必要时启动修订，维护章程严肃性与赛事公信力。

3. 底线不变：修订不得违背赛事公平公正等核心原则，不得损害参赛主体与社区成员合法权益。



#### 修订发起

满足以下任一条件，可启动章程修订提案：

1. 界面生态运营团队或大赛组委会三分之一及以上成员联名发起；

2. 经由界面生态运营团队或大赛组委会检查、用户反馈发现章程内部分内容存在漏洞、缺陷，影响活动正常开展、违背赛事核心原则；

3. 赛事开展的环境、需求发生较大变化，现有章程无法覆盖核心事项，确需调整完善。



#### 民主审议程序

章程修订须严格履行民主程序，由界面生态运营团队与大赛组委会共同组织审议：

1. 草案形成：草案由界面生态运营团队、大赛组委会结合已有的赛事举办经验、社区意见、用户反馈等内容共同修订。

2. 审议表决：草案形成后，由界面生态运营团队代表、大赛组委会代表共同组成审议会议，参会人员三分之二及以上表决同意方可通过。草案一经通过即成为正式章程，产生效力。



### 章程解释与规则监督

1. 章程解释须严格遵循章程核心原则与条文本意，不得作出违背章程精神的扩大或限缩解释。

2. 界面生态运营团队与大赛组委会共同对赛事全周期规则执行情况进行监督，接受社区成员对规则文件违规情形的异议反馈，按本章程异议处理机制核实处理。

3. 本章程的最终解释权归界面生态与大赛组委会所有。



### 配套文件校验标准

所有赛事配套文件的制定与发布，均需符合以下校验标准：

1. 章程一致性标准：文件内容不得与本章程的条文规定、基本原则与核心精神相抵触，不得变更章程设定的核心规则，仅可在章程框架内对执行细节、操作流程进行细化与补充。

2. 公平公正性标准：文件规则需对所有参赛主体一视同仁，不得设置歧视性条款，不得针对特定主体放宽或收紧标准；涉及评审、投票、获奖的规则需清晰透明，可接受社区全员监督。

3. 公益属性标准：文件内容不得涉及任何报名、评审、参赛等收费事项，不得绑定商业合作、商业变现内容，不得将赛事作品、赛事资源用于商业盈利目的。

4. 口径统一性标准：文件中涉及赛事规则、时间节点、奖项设置、权益说明等核心信息的表述与相关专用术语的使用，需与本章程及已发布的正式文件保持一致，不得出现前后矛盾、用语混乱、表述歧义等问题，避免造成公众认知错乱与规则曲解。

5. 合规保密标准：文件内容需符合法律法规与社区规则，不得侵犯他人合法权益；涉及评审非公开信息、参赛选手个人隐私的内容，需严格遵守保密与隐私保护要求，不得违规公开。

### 配套文件审核发布流程

所有正式对外发布的配套文件，均需履行以下全流程审核，未经审定的文件不得对外发布：

1. 起草环节：由大赛组委会根据赛事执行需求起草文件初稿，明确文件的适用范围、核心内容与执行节点。

2. 内部初审：文件初稿提交大赛组委会负责人进行初审，重点核查内容合规性、完整性、逻辑严谨性与执行可操作性，出具初审修改意见，起草组据此调整完善。

3. 终审确认：经初审通过的文件，由大赛组委会与界面生态运营团队共同终审确认；涉及评审规则、奖项核算等评审相关事项的文件，需同步征询评审委员会代表的专业意见。

4. 发布归档：终审通过的文件，由大赛组委会通过官方渠道统一发布；文件原稿与终稿同步归档留存，保障全程留痕可追溯。

5. 纠错回溯：文件发布后若发现存在表述歧义、与章程不符或规则漏洞，大赛组委会需立即发布更正说明，同步启动回溯修订流程，按上述审核程序重新审定后发布。

# 大赛宣传及社区推广



### 宣传渠道与基本原则

赛事宣传主要在社区内部的官方账号（IPOA 赛事官方帐号）与外部主流社媒平台开展，宣传形式上需要风格统一、形式多样，鼓励对于宣传形式的创新；内容则须符合赛事定位、有关文件规定与章程核心精神，真实客观、正向引导，不发布与赛事规则不符的夸大性、误导性信息。 

### 赛前宣传

大赛启动前，官方账号在社区内部发布大赛预热信息、大赛正式公告、大赛操作指南、信息公开目录等内容，保障参赛者对于赛事信息的快速获取；同步面向社区内外的赛事受众，制作发布宣传海报、短视频等宣传资料，扩大赛事覆盖面，吸引 PPT OS 设计爱好者与社区新老成员参与。

### 赛中宣传

1. 报名及作品提交阶段：在延续赛前宣传的部分常态化宣传任务同时，通过社区官方社群每日推送报名提醒、参赛操作指南，集中解答参赛常见问题，保障选手顺利完成报名提交；

2. 初审、复审阶段：开启 "人气之星" 投票引导，推送赛事进展动态，鼓励社区成员参与互动，提升社区活跃度；

3. 所有赛中宣传内容不得泄露评审未公开信息，不得提前透露候选获奖名单。

### 赛后及后续推广

1. 结果公示阶段：在社区内部发布获奖名单；启动针对获奖选手的专访，分享设计技巧、创作经验、参赛经验与成长感悟；

2. 长期推广：在社区官网开设优秀作品常设展示专区，附上评审点评与设计说明，供社区成员交流学习；可邀请获奖选手参与到每周挑战活动题目制定环节，助力优质设计技巧、创作经验在社区内的宣发推广，助力培育社区优质创作者、积淀高质量内容，营造良好设计交流氛围，增强社区凝聚力；

3. 赛事成果总结内容同步推送至外部主流社媒平台，提升界面生态社区行业知名度与圈子影响力，吸引新成员加入。

### 宣传口径管理机制

1. 责任主体：IPOA 赛事组委会负责人负责审核全渠道宣传内容的合规性校验与口径统一，监督宣传工作的有序开展。

2. 口径管理要求：全平台宣传内容须保持规则表述、时间节点、奖项说明等核心信息一致，不得出现与章程、正式配套文件相悖的内容；针对社区成员的咨询，须以章程与正式发布的规则文件为依据答复，无明确规定的事项须上报组委会后统一答复，严禁个人擅自发布失实的内容。
